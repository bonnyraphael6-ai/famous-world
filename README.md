# Famous World 🌍

A social media Telegram Mini App - Instagram-like platform for sharing content.

## Features
- 📸 Post links from Instagram, TikTok, YouTube, Twitter/X, Facebook
- ❤️ Likes, comments, shares, views
- 👥 Follow/unfollow users
- 💬 Direct messages with voice & video call support
- 🎯 Famous Coins (earned from engagement)
- 🔴 Live streaming
- ✅ Verification badges (blue, gold, purple, green)
- 🤖 Bot engagement system
- 👑 Celebrity accounts (pre-loaded)
- 🛡️ Admin dashboard
- 📱 Telegram Mini App compatible

## Setup

1. Clone this repo
2. Install dependencies: `npm install`
3. Copy `.env` and fill in your Supabase credentials
4. Run the SQL schema in Supabase SQL Editor
5. Start dev server: `npm run dev`

## Deploy

Build: `npm run build`
Deploy the `dist/` folder to any static hosting (Vercel, Render, Netlify).

## Tech Stack
- React 18 + Vite
- Supabase (database + auth)
- React Router v6
- Lucide Icons
